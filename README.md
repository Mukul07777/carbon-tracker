# CarbonSense – Carbon Footprint Awareness Platform

A web app that calculates your daily carbon footprint based on transport, diet, and energy usage, then generates personalised AI-powered reduction tips using Claude (Anthropic).

## Features
- Real-time CO₂ calculator with IPCC-aligned emission factors
- AI-generated personalised tips (Claude Sonnet)
- Fully accessible (ARIA labels, keyboard navigation, skip links)
- India-specific recommendations and benchmarks
- Responsive design
